JS and jQuery Vocab

Below is a list of vocab relating to JS and jQuery. Click a word to switch it to the definition, click again to switch back to the word.

  • JavaScript A scripting language for webpages.
  • Operators Symbols used to perform operations in JavaScript (e.g. with '1 + 1', + is the operator).
  • Variable Contains data values.
  • Variable Naming Conventions Use camel case, capitalizing the first letter of every word with no spaces in between.
  • Function A block of code that does something when called on.
  • Method A method can be called on an object to manipulate it.
  • Argument Arguments pass data to functions.
  • Parameter Parameters are defined for functions inside of their parentheses, they are names for values that will be passed in by arguments.
  • Return Indicates that a value should be returned from the following operation.
  • Chaining methods Calling a method on a previous method to modify the previous method's output.
  • Strings A data type indicating text.
  • Booleans A data type that is set to either true or false.
  • Undefined A data type meaning there is no value.
  • NaN A type of integer meaning Not a Number.
  • Escape A way to literalize a special character by putting a backslash in front of it (e.g. \*)
  • Alert A method that will pop up a window containing the data in its parentheses.
  • Comments Used to add documentation within the code.
  • jQuery A JavaScript library that has predefined functions and animations, among other things, that make it so the developer doesn't have to create them manually.
